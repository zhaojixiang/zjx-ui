## Clock

Demo:
```tsx
import React from 'react';
import { Clock } from 'zjx-ui';

export default () => <Clock />;
```

矩形填充 描边 擦除:
| 属性 | 描述 |
| --- | --- |
| fillStyle | 填充颜色 |
| fillRect | 矩形填充 |
| strokeStyle | 描边 |
| strokeRect | 矩形描边 |
