## 面试题

#### [] 空数组转换

1. 直接用 if 判断，[]会被转换成 true；
2. 将[]与布尔值进行比较，[]和布尔值都会被转换成 Number，因此[]会被转换成 0，false 也会被转换成 0，因此[] == false;
